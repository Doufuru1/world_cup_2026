# World Cup 2026 - 链上世界杯赛事金融协议

基于 FIFA 世界杯（48队 / 104场比赛）构建的链上资金收敛系统。

## 项目结构

```
world_cup_2026/
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── app.js          # 交互逻辑
└── images/             # 图片资源
```

## 核心功能

- **首页**: 赛事大厅、总奖池、倒计时
- **赛事预测**: 104场比赛、胜平负预测、赔率系统
- **冠军池**: 锁仓系统、权重机制、结算分配
- **数据看板**: 协议统计、资金飞轮、权重公式

## 技术栈

- HTML5 / CSS3
- JavaScript (ES6+)
- Web3.js (钱包连接)

## 部署

项目部署在 Vercel: https://world-cup-2026.vercel.app

## Tokenomics

- 3% 交易税
  - 1% 回购销毁
  - 1% 比赛奖池
  - 1% 冠军池基金

## 锁仓机制

| 锁仓时长 | 赎回费率 | 权重倍数 |
|---------|---------|---------|
| 1-5天   | 6%      | 1x      |
| 6-10天  | 5%      | 2x      |
| 11-20天 | 3-4%    | 5x      |
| 21-30天 | 2%      | 7x      |
| 31-40天 | 1-1.5%  | 10x     |

## 冠军池分配

- 冠军用户: 70%
- 回购销毁: 20%
- 再投资: 10%
